## ✨ Key Features

- **Global File Indexing**: Search for specific file types (APK, EXE, PDF, ZIP, etc.) across the millions of public repositories on GitHub.
- **Instant Preview**: Visual fade-in previews for images, icons, and textures.
- **Smart Filtering**: Filter by file type, repository author, language, and recency.
- **Local Favorites**: Save assets to your local device for quick access anytime.
- **Productivity Shortcuts**:
  - `/` to focus search.
  - `[` Grid View.
  - `]` List View.
  - `Esc` Close all.
- **Premium UI**: Ultra-dark theme with glassmorphism, noise overlays, and smooth animations.

## 🛠️ Getting Started

1. **Obtain a PAT**: You need a GitHub Personal Access Token (classic or fine-grained) with `read` permissions.
2. **Setup**: Paste your token into the security field in the navbar.
3. **Search**: Use keywords or GitHub search syntax like `filename:app.apk` or `extension:exe`.

## 🔒 Privacy & Security

AssetHub runs entirely in your browser. Your GitHub Token is stored locally in `localStorage` and is never sent to any server other than GitHub's official API.
